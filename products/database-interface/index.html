Database Interface

VISTA may be accessed by any application with ODBC support. This allows users to operate on VISTA data from Microsoft Access, or query results directly into an Excel spreadsheet. Our web interface also allows reports, data tables, and query results to be downloaded in a format compatible with Microsoft Excel, so they may be analyzed in any way the user desires.
